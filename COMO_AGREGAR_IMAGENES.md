# Cómo agregar las imágenes del logo

## Ubicación de las imágenes

Coloca las imágenes en la siguiente carpeta:

```
client/public/images/
```

## Archivos necesarios

1. **Escudo**: `escudo.png`
   - Ruta completa: `client/public/images/escudo.png`

2. **Logo de texto**: `logo-texto.png`
   - Ruta completa: `client/public/images/logo-texto.png`

## Pasos

1. Abre la carpeta `client/public/images/`
2. Coloca tus imágenes con los nombres exactos:
   - `escudo.png`
   - `logo-texto.png`
3. Si tus imágenes tienen otros nombres, avísame y actualizo las rutas en el código

## Verificación

Una vez que agregues las imágenes, recarga la página en el navegador y deberías ver tus logos reales en lugar de los placeholders.
