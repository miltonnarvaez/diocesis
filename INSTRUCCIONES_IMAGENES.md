# Instrucciones para Agregar Imágenes

## Ubicación de Imágenes

Las imágenes deben colocarse en la carpeta `client/public/images/`

## Imágenes Requeridas

### 1. Escudo del Municipio
- **Archivo**: `escudo.png`
- **Ubicación**: `client/public/images/escudo.png`
- **Recomendación**: Imagen PNG con fondo transparente, mínimo 200x200px

### 2. Bandera del Municipio
- **Archivo**: `bandera.png`
- **Ubicación**: `client/public/images/bandera.png`
- **Recomendación**: Imagen PNG, proporción 3:2 (ancho:alto), mínimo 300x200px

## Nota

Si las imágenes no están disponibles, el sistema mostrará automáticamente versiones SVG del escudo y la bandera como respaldo.

## Imágenes Dinámicas

Las imágenes de noticias y convocatorias se cargan desde la base de datos a través del campo `imagen_url`. Estas imágenes pueden ser:
- URLs externas
- Rutas relativas a archivos subidos al servidor

Para subir imágenes al servidor, usa el panel de administración en `/admin`.
