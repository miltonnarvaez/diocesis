# Instrucciones para la Imagen del Hero

## Ubicación de la Imagen

Coloca la imagen en:
```
client/public/images/hero-grupo-personas.jpg
```

## Especificaciones de la Imagen

- **Nombre del archivo**: `hero-grupo-personas.jpg` (o `.png`, `.webp`)
- **Formato recomendado**: JPG de alta calidad o PNG
- **Dimensiones recomendadas**: 
  - Ancho: 1920px - 3840px (alta resolución)
  - Alto: 400px - 500px (más ancha que alta)
  - Ratio: aproximadamente 4:1 o 5:1 (ancho:alto)
- **Tema**: Grupo de personas (representando comunidad, participación ciudadana, concejo)
- **Calidad**: Alta resolución para que se vea bien en pantallas grandes

## Formatos Alternativos

Si tu imagen tiene otro nombre o formato, puedes:
1. Renombrarla a `hero-grupo-personas.jpg`
2. O avisar el nombre exacto y actualizo el código

## Fallback

Si la imagen no existe, se mostrará un fondo verde con gradiente.
